# Minimalist Carbon Red - Loading Screen (MP9 Style)

Acest loading screen a fost conceput special pentru cei care isi doresc un design curat, rapid si de impact pentru serverul lor de FiveM. Este optimizat pentru a rula fluid pe orice rezolutie si pune accent pe identitatea vizuala a serverului.

Proiectul a fost creat initial pentru comunitatea MP9, dar este acum disponibil public pentru oricine doreste sa il foloseasca sau sa il imbunatateasca.

## Caracteristici principale
---
* **Animatii Premium**: Logo-ul beneficiaza de un efect de intrare cinematic (flash/scale) si un breathing glow rosu care atrage atentia imediat.
* **Design Minimalist**: Interfata a fost curatata de elemente inutile, lasand doar logo-ul central si elementele esentiale de control.
* **Player Muzical Integrat**: Include afisarea numelui piesei, slider de volum si controale complete de redare.
* **Bara de Progres Sincronizata**: O linie rosie distincta la baza ecranului care pulseaza si urmareste progresul incarcarii.
* **Suport Video**: Permite utilizarea oricarui fisier MP4 ca fundal.

## Instructiuni de Instalare
---
1. Descarcati arhiva si extrageti folderul in directorul `resources` al serverului.
2. Redenumiti folderul in `mp9_loadingscreen`.
3. Adaugati `ensure mp9_loadingscreen` in fisierul dumneavoastra `server.cfg`.
4. Plasati logo-ul serverului in folderul `logo` sub numele `logo.png`.

## Personalizare
---
* **Logo**: Inlocuiti `logo/logo.png`. Se recomanda formatul PNG transparent (512x512px).
* **Video**: Inlocuiti `video/background.mp4` cu clipul dorit.
* **Culoare**: Tema poate fi schimbata modificand variabila `--primary` in fisierul `css/style.css`:
```css
:root {
    --primary: #ff3333; /* Codul de culoare dorit */
}
```

## Control tastatura
---
* **SPACE**: Play / Pauza muzica.
* **Sageata SUS / JOS**: Ajustare volum.
* **Sageata STANGA / DREAPTA**: Schimbare piesa.
